# JLPT HVPT Trainer

一个面向日语 JLPT 词汇的非官方训练网站。它把词汇记忆拆成“看图联想”和“听音辨认”两条线，用多单词、多声音、即时反馈的方式，帮助学习者更快区分容易混淆的日语发音、写法和意义。

## 核心功能

- 覆盖 JLPT N1-N5 词汇练习，当前包含 283 道看图题和 2257 道听音/文字题。
- 提供图片、音频、释义三种输入路径，让词汇不是只靠死背单词表。
- 每次作答后立即显示正确答案，适合短时间反复练习。
- 支持中文、英文、日文界面切换，方便不同阶段使用。
- 支持按等级、类别、写法筛选，包括平假名、片假名、汉字/混合写法。
- 支持本地静态运行和 GitHub Pages 发布，不需要后端服务器。

## 训练模式

### 看词选图

看到一个日语单词，从多张图片中选择最相关的一张。这个模式适合建立“单词 -> 具体画面”的联想，尤其适合名词、物品、地点、人物、食物等有明确视觉对象的词。

### 听音选词

先听日语音频，再从多个候选词中选择正确答案。这个模式重点训练听辨能力，适合区分读音接近、假名相似、长短音容易混淆的词。

### 听音写词

听到日语音频后，直接输入单词或读法。相比选择题，这个模式更接近真实回忆，可以检查自己是否真的听懂并能主动写出答案。

### 看义写词

看到词义提示后，输入对应的日语单词或读法。这个模式适合从中文/英文释义反推日语表达，帮助把“认识”推进到“能想起来”。

## 音频训练

听音练习使用 VOICEVOX 合成的日语音频，并预生成到网站静态资源中。训练时可以：

- 使用随机角色播放，增加声音变化。
- 手动选择固定角色，集中熟悉某一种声音。
- 在听音选词和听音写词中反复播放音频。

这种设计更接近 HVPT（High Variability Phonetic Training，高变异语音知觉训练）的思路：不要只听一个声音、一个词，而是在不同声音和不同单词之间反复辨认，并马上得到反馈。

## 网站优点

- **低门槛**：打开网页即可练习，不需要登录账号。
- **反馈快**：选择或输入后立即显示答案，减少“猜完不知道对不对”的空档。
- **可离线**：数据、图片、音频都在静态文件中，本地启动后可以离线使用。
- **隐私友好**：练习过程不上传服务器，也不收集用户数据。
- **适合扩展**：数据、图片和音频分离，之后可以继续加入更多例句、题型、错题复习和间隔重复。
- **来源透明**：词汇、释义、例句、图片和音频来源集中写在来源与许可页面中。

## 本地运行

进入项目后打开：

```text
hvpt_site/start_site.bat
```

脚本会启动一个本地静态服务器，然后访问网站首页。首页是介绍页，点击“开始训练”会进入：

```text
hvpt_site/trainer.html
```

不建议直接双击 `index.html`，因为浏览器可能会拦截本地 JSON 数据读取。

## 分享给别人

如果只是让别人体验网站，建议只打包这些内容：

- `hvpt_site`
- `README.md`
- `NOTICE.md`
- `LICENSE`

对方解压后进入 `hvpt_site`，双击 `start_site.bat` 即可打开网站。对方不需要安装 ComfyUI，也不需要安装生成图片或生成语音的模型；网站运行只需要已经生成好的静态资源。

- 例句数据来自 Tatoeba 及本地整理内容，需要保留对应来源说明。
